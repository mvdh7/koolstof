Miscellaneous tools for marine carbonate chemistry and other such things.

Highly unstable and unreliably documented.

**Install:**

    pip install koolstof

**Import:**

```python
import koolstof as ks
```

**Find current version:**

```python
ks.say_hello()
```

**Citation:**

> Humphreys, M. P. (2020).  koolstof: miscellaneous Python tools for marine carbonate chemistry.  *Zenodo*.  [doi:10.5281/zenodo.3999292](https://doi.org/10.5281/zenodo.3999292).
